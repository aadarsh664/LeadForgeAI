Objective

Create the permanent Design System for LeadForgeAI.

After this task, every screen must use reusable UI components.

No page should contain custom one-off UI elements.

---

Problem

The application currently contains individual UI components.

Without a Design System the UI will become inconsistent.

---

Goal

Create a reusable component library.

Create global design tokens.

Create standardized buttons, cards, inputs and typography.
