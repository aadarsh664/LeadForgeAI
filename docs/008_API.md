# LeadForgeAI — API Architecture

Version: 1.0.0

Status: Draft

---

# Purpose

This document defines the API architecture of LeadForgeAI.

Every communication between the Desktop Application, Backend, n8n and future integrations must follow this specification.

The API is the only official communication layer of the platform.

---

# API Philosophy

Everything communicates through APIs.

No module should directly access another module's internal implementation.

Every request should pass through the Backend.

---

# Communication Flow

↓

↓

↓

# External Communication
