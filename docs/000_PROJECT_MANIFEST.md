# LeadForgeAI — Project Manifest

Version: 1.0.0

Status: Planning Phase

---

# Mission

LeadForgeAI is a self-hosted desktop application that helps users discover, enrich, organize and automate business and professional leads using public business information and AI-assisted workflows.

The system is designed for non-technical users.

The user should only need to:

1. Start Docker Desktop
2. Open LeadForgeAI
3. Click the Power Button
4. Wait until every service becomes healthy
5. Start using the application

No command line should be required during normal usage.

---

# Vision

Create the most powerful self-hosted Lead Intelligence Platform.

Everything should run locally whenever possible.

The software must be modular.

Every module should work independently.

Every module should communicate through APIs.

The application should remain usable even if one module fails.

# Development Rules

Never start coding before documentation.

Never modify architecture without documentation.

Never change database structure without documentation.

Never break existing APIs.

Every feature requires:

- Specification
- Task
- Review
- Documentation
- Implementation
- Testing

---

# Software Philosophy

LeadForgeAI is NOT a scraper.

LeadForgeAI is a Lead Intelligence Platform.

Scraping is only one component.

The platform should continuously enrich existing data instead of repeatedly collecting the same information.
