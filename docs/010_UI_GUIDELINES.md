# Purpose

This document defines the visual design language of LeadForgeAI.

Every screen, component and interaction must follow these guidelines.

---

# Design Philosophy

Minimal

Modern

Fast

Professional

No unnecessary visual clutter.

---

# User Experience Goals

A beginner should understand the interface within five minutes.

The application should reduce cognitive load.

Every important action should require as few clicks as possible.
